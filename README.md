# Web Application (EAT with AI)

## Overview
This is the web application repository of the Event Attendance Tracking Software. The goal of building this application is to keep track of attendance in an event and provide live updates. This repository contains code that sets up the entire web application.
Tech Stack:
* Material UI (MUI) Library
* React.js
* JavaScript
* HTML/CSS

## Setup Instructions
To successfully setup the web application to run on your local machine, please install the following dependencies:
* [Node.js](https://nodejs.org/en/download)

First, `git clone` this repository to access all the code. Then, to install all the dependencies run `npm install`.

Also, create `.env` file. It will contain the the following line `DANGEROUSLY_DISABLE_HOST_CHECK=true`.

## Running the application
To run the web application, `cd` into the `webApp` directory and use the `npm start` command. This will launch the application in your default browser.
